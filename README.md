📊 Spending Tracker
A full-stack Java Spring Boot application that helps users track, manage, and analyze their daily expenses with secure authentication and categorized insights.

🚀 Features
🔐 User Authentication
JWT-based authentication
Google OAuth login
💸 Expense Management
Add, update, delete expenses
Categorize expenses (Food, Travel, Shopping, etc.)
📊 Insights & Analytics
Monthly expense calculation
Highest expense tracking
📧 Email Integration
Gmail SMTP support (notifications / password reset)
🌐 Full-Stack Deployment
Frontend served via Spring Boot (resources/static)
Backend APIs integrated

🛠️ Tech Stack
Backend: Java 17, Spring Boot
Database: MongoDB Atlas
Security: Spring Security, JWT, Google OAuth
Email Service: Gmail SMTP
Deployment: Render
Frontend: HTML, CSS, JavaScript
